================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "CORS error" in browser console
SOLUTION:
1. Check CORS_ORIGINS in Render exactly matches your Vercel URL
2. Must include https://
3. No trailing slash
4. Click Save in Render and wait for redeploy
5. Clear browser cache (Ctrl+Shift+Delete)
6. Hard refresh (Ctrl+Shift+R)

PROBLEM: "Cannot connect to API"
SOLUTION:
1. Check REACT_APP_BACKEND_URL in Vercel matches Render URL
2. Verify Render service status is "Live"
3. Visit https://restaurant-api.onrender.com/api in browser
4. Should show: {"message": "Hello World"}
5. Check Render logs for errors

PROBLEM: "Payment not working"
SOLUTION:
1. Check RAZORPAY_KEY_ID starts with rzp_test_ or rzp_live_
2. Verify both KEY_ID and KEY_SECRET are set
3. Check Razorpay account is verified
4. Try test mode first (rzp_test_*)
5. Check Razorpay script is loading (Network tab)

PROBLEM: "Admin panel won't load"
SOLUTION:
1. Check frontend deployed successfully (Vercel shows Ready)
2. Try accessing from incognito/private mode
3. Clear browser cookies and cache
4. Check /admin route exists
5. Verify no 404 errors in Network tab

PROBLEM: "Database connection error"
SOLUTION:
1. Verify MONGO_URL is correct
2. Check MongoDB IP whitelist
3. Verify password is correct (special chars might need escaping)
4. Try connecting locally to verify connection string
5. Check MongoDB cluster status (should be "Active")
